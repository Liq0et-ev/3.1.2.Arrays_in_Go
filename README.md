# 3.1.2.Arrays_in_Go
# Divu virkņu salīdzināšana 

Šī Go programma nolasa no ievades divas veselu skaitļu virknes. Ir zināms, ka katra virkne beidzas ar skaitli 0, un šī nulle kalpo kā beigu marķieris (netiek uzskatīta par virknes elementu). <br>

Pēc ievades programma salīdzina abas virknes un izvada visus pirmās virknes elementus, kuriem otrajā virknē nav neviena vienāda elementa. Rezultātā tiek parādīti tikai tie pirmās virknes skaitļi, kas otrajā virknē neparādās. <br>
